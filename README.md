# ФОС по дисциплине «Методы Монте-Карло в машинном обучении»

**Назначение:** Комплексный фонд оценочных средств для оценивания проектной работы студентов по курсу.

**Целевая аудитория:** Преподаватели, разрабатывающие и проводящие курс; студенты, выполняющие проекты.

**Модель компетенций:** Компетентностно-ролевая модель ИТМО (КРМ), роли: ML Engineer, Data Analyst, ML Researcher.

**Шкала оценивания:** 10-балльная, две оси: «Программное решение (MVP)» и «Документация».

**Основное содержание курса (по учебнику Войтишека А.В. «Лекции по численным методам Монте-Карло»):**

- **Генерация случайных чисел** (раздел 9): линейные конгруэнтные генераторы, мультипликативный метод вычетов, тестирование генераторов (спектральный тест, хи-квадрат, тесты на равномерность и независимость).
- **Моделирование случайных величин и векторов** (разделы 2, 10–11): метод обратной функции распределения, мажорантный метод исключения (метод Неймана), метод дискретной суперпозиции, метод Уолкера, моделирование двумерных и многомерных векторов с зависимыми компонентами.
- **Оценка интегралов методом Монте-Карло** (разделы 1–5): весовой оцениватель, методы уменьшения дисперсии (выборка по важности, стратифицированная выборка, метод выделения главной части, метод интегрирования по части области).
- **Цепи Маркова и решение интегральных уравнений Фредгольма II рода** (разделы 6–8): марковские интегральные уравнения, ряд Неймана, основной весовой оцениватель (оценка по столкновениям), оцениватель по поглощениям, метод сопряжённых блужданий.
- **Квази-Монте-Карло** (раздел 9.7): последовательности Соболя, Холтона, Ван дер Корпута, сравнение с псевдослучайными последовательностями.
- **Моделирование случайных процессов** (раздел 13): броуновское движение, пуассоновские процессы, гауссовские процессы, спектральные модели однородных случайных полей.

**Связь с машинным обучением:** методы Монте-Карло лежат в основе байесовского вывода (MCMC), обучения с подкреплением (MCTS), оценки неопределённости в нейросетях (MC Dropout) и генеративных моделей (VAE). Курс даёт фундаментальную математическую базу для этих приложений.

**Быстрый старт:** См. `release/how_to_apply.md`.
