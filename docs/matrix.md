## 4. Этапы формирования компетенций и связь с 10-балльной шкалой

| Этап (уровень освоения) | Содержание этапа | Основные оценочные средства | Баллы |
|-------------------------|------------------|------------------------------|-------|
| **Начальный (исполнитель)** | Освоение инструментов по инструкции: базовые алгоритмы (генерация случайных чисел, оценка π), работа с готовыми библиотеками. | Лабораторные работы (ОС-1), Тесты (ОС-2) | 4–5 |
| **Базовый (специалист)** | Самостоятельная реализация алгоритмов: MCMC, Метрополис-Гастингс, Гиббс, валидация сходимости. | Лабораторные работы (ОС-1), Peer-review (ОС-3) | 6–7 |
| **Продвинутый (ведущий специалист)** | Проектирование и оптимизация: HMC/NUTS, MCTS, байесовская оптимизация, сравнение методов. | Курсовой проект (ОС-4), Защита (ОС-5) | 8–9 |
| **Экспертный (архитектор)** | Архитектурная и стратегическая деятельность: разработка новых подходов, интеграция методов в комплексные системы. | Курсовой проект (ОС-4), Защита (ОС-5) | 10 |

## 5. Ролевые портреты уровней 4 / 6 / 8 / 10

### 5.1. ML Engineer (Инженер машинного обучения)

| Балл | Портрет уровня |
|------|----------------|
| **10** | Формирует стратегию применения Монте-Карло, разрабатывает новые алгоритмы сэмплирования, стандартизирует практики MLOps для стохастических моделей. |
| **8** | Проектирует пайплайны байесовского вывода с HMC/NUTS, оптимизирует инференс, обосновывает выбор методов с учётом производительности. |
| **6** | Самостоятельно реализует алгоритмы MCMC (Метрополис-Гастингс, Гиббс), валидирует сходимость, применяет MC Dropout для оценки неопределённости. |
| **4** | Реализует базовые алгоритмы (генерация случайных чисел, оценка π, простая регрессия) по готовым шаблонам. |

### 5.2. Data Analyst (Аналитик данных)

| Балл | Портрет уровня |
|------|----------------|
| **10** | Формирует аналитическую стратегию для стохастических моделей, стандартизирует практики оценки неопределённости и визуализации. |
| **8** | Проектирует эксперименты с использованием байесовских методов, обосновывает выбор метрик (HDI, R-hat, ESS), интерпретирует результаты. |
| **6** | Самостоятельно проводит EDA, проверяет гипотезы о распределениях, строит визуализации сходимости MCMC. |
| **4** | Проводит базовый EDA по шаблону, строит стандартные графики (гистограммы, scatter plot). |

### 5.3. ML Researcher (Исследователь в области МО)

| Балл | Портрет уровня |
|------|----------------|
| **10** | Разрабатывает новые теоретические основы для методов Монте-Карло, публикует результаты, обосновывает сходимость и сложность алгоритмов. |
| **8** | Применяет продвинутый математический аппарат для анализа сходимости MCMC, сравнивает байесовский и частотный подходы. |
| **6** | Демонстрирует понимание ЗБЧ, ЦПТ, условия детального баланса, использует статистические критерии для валидации моделей. |
| **4** | Знает основные теоремы и может объяснить базовые принципы работы алгоритмов Монте-Карло. |
