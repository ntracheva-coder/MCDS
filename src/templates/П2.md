# Цель занятия

Научиться применять аналитические методы построения алгоритмов генерации случайных величин: метод обратной функции, метод суперпозиции и мажорантный метод исключения. Студенты должны уметь находить функции распределения, обратные функции, маргинальные плотности, подбирать мажоранты и оценивать трудоёмкость.

---

# Задания

## Раздел A. Метод обратной функции распределения

Для каждой плотности постройте формулу метода обратной функции (т.е. выразите $u = F^{-1}(r)$, где $r \sim U(0,1)$) и запишите алгоритм генерации $\xi_0$.

1. $f_\xi(u) = 3 \sin 2u \cdot \sin^4 u, \quad 0 < u < \frac{\pi}{2}.$

2. $f_\xi(u) = 3 \sin 2u  \cdot \cos^4 u, \quad 0 < u < \frac{\pi}{2}.$  
   Сравните моделирующие формулы для плотностей A1 и A2.

3. $f_\xi(u) = \frac{5 \ln^4(5u)}{u}, \quad \frac{1}{5} < u < \frac{e}{5}.$

4. $f_\xi(u) = \frac{4u^3}{\sqrt{4u^4 + 9}}, \quad 0 < u < \sqrt{2}.$

---

## Раздел Б. Моделирование двумерного и трёхмерного случайных векторов

Для каждой плотности найдите маргинальные и условные плотности и предложите алгоритм генерации.

1. $f_{(\xi,\eta)}(u,v) = \frac{e^{-4uv}}{v^4}, \quad u > 0, \quad v > \frac{1}{2}.$

2. $f_{(\xi,\eta)}(u,v) = 3 u^2 (u+1) v^u e^{-u^3}, \quad u > 0, \quad 0 < v < 1.$

3. $f_{(\xi,\eta)}(u,v) = \frac{3 v \sin v \cdot e^{-3uv}}{1 - e^{-v}}, \quad 0 < u < \frac{1}{3}, \quad 0 < v < \frac{\pi}{2}.$

4. **(Трёхмерный вектор)**  
   Пусть $(\xi,\eta,\zeta)$ имеет плотность  
   $f_{(\xi,\eta,\zeta)}(u,v,w) = C \cdot u e^{-u} e^{-u v} e^{-w/u}, \quad u>0,\ v>0,\ w>0,$  
   где $C$ — нормировочная константа.

   1. Найдите константу $C$.
   2. Найдите маргинальную плотность $f_\xi(u)$.
   3. Найдите условные плотности $f_{\eta|\xi}(v|u)$ и $f_{\zeta|\xi}(w|u)$.
   4. Запишите пошаговый алгоритм генерации тройки $(\xi_0,\eta_0,\zeta_0)$.

5. **(Трёхмерный вектор с зависимостью)**  
   Пусть $f_{(\xi,\eta,\zeta)}(u,v,w) = C \cdot e^{-(u+v+w)} (1 + u v w), \quad u>0,\ v>0,\ w>0,$  
   где $C$ — константа нормировки.

   1. Найдите $C$.
   2. Покажите, что компоненты не являются независимыми.
   3. Найдите условную плотность $f_{\zeta|\xi,\eta}(w|u,v)$.
   4. Предложите алгоритм генерации, используя метод суперпозиции для моделирования пары $(\xi,\eta)$ с последующей генерацией $\zeta$ из условного распределения.

---

## Раздел В. Методы суперпозиции

Для каждой плотности сформулируйте метод дискретной или интегральной суперпозиции, запишите алгоритм генерации и оцените трудоёмкость (если применимо).

1. **(Интегральная суперпозиция)**  
   $f_\xi(u) = \int_{1/2}^{+\infty} \frac{e^{-4uv}}{v^4} \, dv, \quad u > 0.$  
   Сравните полученный алгоритм с алгоритмом из пункта Б1.

2. **(Дискретная суперпозиция)**  
   $f_\xi(u) = \frac{\sin u}{2} + \frac{\cos u}{2}, \quad 0 < u < \frac{\pi}{2}.$

3. $f_\xi(u) = \frac{3 \cos \sqrt{u}}{8\sqrt{u}} + \frac{1}{\pi^2}, \quad 0 < u < \frac{\pi^2}{4}.$

4. $f_\xi(u) = e^{-2u} + e^{-3u} + e^{-6u}, \quad u > 0.$

---

## Раздел Г. Мажорантный метод исключения

Для каждой плотности (заданной с точностью до нормировки) подберите мажоранту $g(x)$, найдите константу $M$ и оцените теоретическую вероятность принятия $1/M$.

1. $g(u) = \left(2 + \frac{\arcsin u}{5\pi}\right) u^3, \quad 0 < u < 1.$

2. $g(u) = \frac{1}{u^2 \lg(u+10) + 10}, \quad 0 < u < 1.$

3. **(Двумерный случай)**  
   Постройте алгоритм численного моделирования выборочного значения $\xi_0 = (\nu_0, \mu_0)$ двумерного случайного вектора $\xi = (\nu, \mu)$, плотность которого пропорциональна  
   $g(u,v) = u^2 v^3 \cos v^3 \, e^{-v u^3} \left(1 + \frac{11}{25} \cos^2(u+v)\right), \quad u > 0, \quad 0 < v < \sqrt[3]{\frac{\pi}{2}}.$  
   Оцените сверху трудоёмкость метода.

     1. Проверьте нормировку.  
     2. Найдите маргинальные распределения $\xi$, $\eta$ и условное $\zeta|\xi$.

## Задачи по мажорантному методу исключения (продолжение)

Д13. Постройте алгоритм для двумерной плотности, пропорциональной

      Оцените трудоёмкость.

5. По окончании занятия проводится устное обсуждение решений и разбор сложных моментов.
