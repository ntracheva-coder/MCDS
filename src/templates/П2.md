# Цель занятия {#цель-занятия .unnumbered}

Научиться применять аналитические методы построения алгоритмов генерации
случайных величин: метод обратной функции, метод суперпозиции и
мажорантный метод исключения. Студенты должны уметь находить функции
распределения, обратные функции, маргинальные плотности, подбирать
мажоранты и оценивать трудоёмкость.

# Задания {#задания .unnumbered}

## Раздел A. Метод обратной функции распределения {#раздел-a.-метод-обратной-функции-распределения .unnumbered}

1.  $$f_\xi(u) = 3 \sin 2u \, \sin^4 u, \quad 0 < u < \frac{\pi}{2}.$$

2.  $$f_\xi(u) = 3 \sin 2u \, \cos^4 u, \quad 0 < u < \frac{\pi}{2}.$$
    Сравните моделирующие формулы для плотностей $A_1$ и $A_2.$

3.  $$f_\xi(u) = \frac{5 \ln^4(5u)}{u}, \quad \frac{1}{5} < u < \frac{e}{5}.$$

4.  $$f_\xi(u) = \frac{4u^3}{\sqrt{4u^4 + 9}}, \quad 0 < u < \sqrt{2}.$$

## Раздел Б. Моделирование двумерного случайного вектора {#раздел-б.-моделирование-двумерного-случайного-вектора .unnumbered}

1.  $$f_{(\xi,\eta)}(u,v) = \frac{e^{-4uv}}{v^4}, \quad u > 0, \quad v > \frac{1}{2}.$$

2.  $$f_{(\xi,\eta)}(u,v) = 3 u^2 (u+1) v^u e^{-u^3}, \quad u > 0, \quad 0 < v < 1.$$

3.  $$f_{(\xi,\eta)}(u,v) = \frac{3 v \sin v \, e^{-3uv}}{1 - e^{-v}}, \quad 0 < u < \frac{1}{3}, \quad 0 < v < \frac{\pi}{2}.$$

4.  Рассмотрим трёхмерный случайный вектор $(\xi,\eta,\zeta)$ с
    плотностью
    $$f_{(\xi,\eta,\zeta)}(u,v,w) = C \cdot u e^{-u} e^{-u v} e^{-w/u}, \quad u>0,\ v>0,\ w>0,$$
    где $C$ --- нормировочная константа.

    1.  Найдите константу $C$.

    2.  Найдите маргинальную плотность $f_\xi(u)$.

    3.  Найдите условные плотности $f_{\eta|\xi}(v|u)$ и
        $f_{\zeta|\xi}(w|u)$.

    4.  Запишите пошаговый алгоритм генерации тройки
        $(\xi_0,\eta_0,\zeta_0)$.

5.  Пусть трёхмерный вектор $(\xi,\eta,\zeta)$ имеет плотность
    $$f_{(\xi,\eta,\zeta)}(u,v,w) = C \cdot e^{-(u+v+w)} (1 + u v w), \quad u>0,\ v>0,\ w>0,$$
    где $C$ --- константа нормировки.

    1.  Найдите $C$.

    2.  Покажите, что компоненты не являются независимыми (например,
        проверьте $f_{\xi,\eta}(u,v) \ne f_\xi(u) f_\eta(v)$).

    3.  Найдите условную плотность $f_{\zeta|\xi,\eta}(w|u,v)$.

    4.  Предложите алгоритм генерации, используя метод суперпозиции для
        моделирования пары $(\xi,\eta)$ с последующей генерацией $\zeta$
        из условного распределения.

## Раздел В. Методы суперпозиции {#раздел-в.-методы-суперпозиции .unnumbered}

1.  Сформулируйте метод интегральной суперпозиции и примените его к
    моделированию выборочного значения $\xi_0$ случайной величины $\xi$
    с указанной плотностью. Сравните полученный алгоритм с алгоритмом,
    полученным в пункте **Б5**.
    $$f_\xi(u) = \int_{1/2}^{+\infty} \frac{e^{-4uv}}{v^4} \, dv, \quad u > 0.$$\
    Сформулируйте метод дискретной суперпозиции и модифицированный метод
    суперпозиции и примените их к моделированию выборочного значения
    $\xi_0$ случайной величины $\xi$ с указанной плотностью.

2.  $$f_\xi(u) = \frac{\sin u}{2} + \frac{\cos u}{2}, \quad 0 < u < \frac{\pi}{2}.$$

3.  $$f_\xi(u) = \frac{3 \cos \sqrt{u}}{8\sqrt{u}} + \frac{1}{\pi^2}, \quad 0 < u < \frac{\pi^2}{4}.$$

4.  $$f_\xi(u) = e^{-2u} + e^{-3u} + e^{-6u}, \quad u > 0.$$

## Раздел Г. Мажорантный метод исключения {#раздел-г.-мажорантный-метод-исключения .unnumbered}

1.  $$g(u) = \left(2 + \frac{\arcsin u}{5\pi}\right) u^3, \quad 0 < u < 1.$$

2.  $$g(u) = \frac{1}{u^2 \lg(u+10) + 10}, \quad 0 < u < 1.$$

3.  Постройте алгоритм численного моделирования выборочного значения
    $\xi_0 = (\nu_0, \mu_0)$ двумерного случайного вектора
    $\xi = (\nu, \mu)$, плотность которого пропорциональна функции
    $$g(u,v) = u^2 v^3 \cos v^3 \, e^{-v u^3} \left(1 + \frac{11}{25} \cos^2(u+v)\right), \quad u > 0, \quad 0 < v < \sqrt[3]{\frac{\pi}{2}}.$$
    Оцените сверху трудоёмкость метода.

# Порядок работы {#порядок-работы .unnumbered}

1.  Студенты работают индивидуально или в малых группах (2--3 человека).

2.  На занятии отводится 90 минут.

3.  Преподаватель кратко напоминает теоретические основы каждого метода.

4.  Студенты решают задачи в тетрадях, оформляя выводы формул и
    алгоритмов.

5.  По окончании занятия проводится устное обсуждение решений.
