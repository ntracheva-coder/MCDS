# Теоретическое введение {#теоретическое-введение .unnumbered}

## Линейный конгруэнтный генератор (LCG) {#линейный-конгруэнтный-генератор-lcg .unnumbered}

Один из простейших алгоритмов генерации псевдослучайных чисел задаётся
рекуррентным соотношением $$X_{n+1} = (a X_n + c) \mod m,$$ где:

- $m > 0$ --- модуль;

- $a$ --- множитель ($0 < a < m$);

- $c$ --- приращение ($0 \le c < m$);

- $X_0$ --- начальное значение (seed).

На выходе используют $U_n = X_n / m \in [0,1)$. Период LCG максимален
($=m$) при выполнении условий Халла--Добелла. Для мультипликативного LCG
($c=0$) период не может превышать $m-1$.

## Тест хи-квадрат {#тест-хи-квадрат .unnumbered}

Проверяет гипотезу о равномерном распределении. Интервал $[0,1)$
разбивается на $k$ равных интервалов. Для выборки объёма $n$ вычисляется
$$\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}, \quad E_i = n/k,$$
где $O_i$ --- наблюдаемые частоты. Статистика сравнивается с
$\chi^2_{1-\alpha}(k-1)$.

## Методы генерации распределений {#методы-генерации-распределений .unnumbered}

В ходе работы используются:

1.  **Метод обратной функции**: $X = F^{-1}(U)$.

2.  **Метод суперпозиции**: $f(x) = \sum p_j f_j(x)$.

3.  **Метод исключения (Acceptance-Rejection)**: моделирование случайной
    величины с целевой плотностью $f(x)$ с помощью вспомогательной
    плотности (мажоранты) $g(x)$. Для реализации подбирается константа
    $M \ge 1$, удовлетворяющая условию:
    $$M \cdot g(x) \ge f(x) \quad \text{для всех } x.$$ Величина $1/M$
    определяет теоретическую вероятность принятия сгенерированной точки
    на каждом шаге алгоритма. Алгоритм:

    1.  Генерируем $X \sim g$ и $U \sim U[0,1]$.

    2.  Если $U < f(X)/(M g(X))$, то принимаем $X$, иначе повторяем.

4.  **Метод Бокса--Мюллера**:
    $Z_1 = \sqrt{-2\ln U_1}\cos(2\pi U_2), \quad Z_2 = \sqrt{-2\ln U_1}\sin(2\pi U_2)$.

5.  **Метод Уолкера (alias)**.

## Встроенный генератор numpy.random {#встроенный-генератор-numpy.random .unnumbered}

Использует PCG64 (Permuted Congruential Generator) с хорошими
статистическими свойствами. В работе используется как эталон для
сравнения.

## Бутстрап {#бутстрап .unnumbered}

Метод статистического моделирования, основанный на многократной выборке
с возвращением из исходной выборки для оценки распределения статистик.
