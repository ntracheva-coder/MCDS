1.  Сформулируйте задачу генерации непрерывной случайной величины с
    заданной плотностью распределения $f(x)$. В чём отличие от генерации
    дискретных распределений?

2.  Опишите метод обратной функции (квантильный метод) для непрерывных
    распределений. При каких условиях он применим? Приведите примеры
    распределений, для которых этот метод даёт простую явную формулу
    (экспоненциальное, распределение Коши, Вейбулла).

3.  Для каких непрерывных распределений метод обратной функции неудобен
    или невозможен? Предложите альтернативные подходы.

4.  В чём заключается метод суперпозиции (смесевое представление)?
    Запишите формулу для плотности смеси и алгоритм генерации. Приведите
    пример (смесь экспоненциальных или нормальных распределений).

5.  Опишите метод исключения (rejection sampling) для генерации из
    плотности $f(x)$ с использованием мажоранты $g(x)$. Запишите условие
    мажорирования и алгоритм.

6.  Как выбирается мажоранта $g(x)$ в методе исключения? Какими
    свойствами она должна обладать? Что такое оптимальная мажоранта?

7.  Введите понятие эффективности метода исключения. Как вычисляется
    вероятность принятия? Запишите формулу и объясните, почему она равна
    $1/M$, где $M = \sup f(x)/g(x)$.

8.  Как оценить среднее число итераций метода исключения до получения
    одного принятого значения? Как это связано с константой $M$?

9.  Сравните метод исключения и метод обратной функции по времени
    работы, сложности реализации и универсальности. В каких случаях
    метод исключения предпочтительнее?

10. Опишите методы генерации гамма-распределения
    $\text{Ga}(\alpha, \beta)$:

    1.  для целого $\alpha$ (метод суммирования экспоненциальных);

    2.  для нецелого $\alpha$ (метод исключения, метод Ченга--Фиста).

    Сравните их эффективность для разных значений $\alpha$.

11. Как связано гамма-распределение с бета-распределением? Опишите
    алгоритм генерации бета-распределения через гамма-распределения. В
    чём недостаток этого подхода для больших параметров формы?

12. Опишите метод Бокса--Мюллера для генерации стандартного нормального
    распределения. Выведите формулы и объясните, почему получаются две
    независимые нормальные величины.

13. В чём отличие полярного метода (Марсальи--Брея) от метода
    Бокса--Мюллера? Какое преимущество даёт полярный метод?

14. Опишите приближённый метод генерации нормального распределения на
    основе центральной предельной теоремы (сумма 12 равномерных
    величин). Оцените его точность и сравните с точными методами.

15. Для каждого из методов генерации нормального распределения
    (Бокс--Мюллер, полярный, ЦПТ-аппроксимация) оцените:

    1.  точность (особенно в хвостах);

    2.  вычислительную сложность;

    3.  количество необходимых равномерных чисел.

16. Как влияет выбор метода генерации на качество симуляций в
    приложениях машинного обучения (инициализация весов, стохастический
    градиентный спуск, байесовские методы)? Приведите примеры.

17. Сформулируйте критерии выбора метода генерации для конкретного
    распределения. Какие факторы нужно учитывать: скорость, точность,
    простота реализации, необходимость предварительных вычислений?

18. Для распределения с плотностью $f(x) \propto e^{-x^2}(1+\sin^2(5x))$
    на $[-\pi, \pi]$ предложите метод генерации (обратная функция,
    исключение, суперпозиция) и оцените его эффективность.

19. Опишите, как метод исключения можно адаптировать для генерации из
    распределений с тяжёлыми хвостами (например, распределение
    Стьюдента). Какие сложности возникают при выборе мажоранты?

20. В чём заключается метод «выборка по важности» (importance sampling)
    и чем он отличается от метода исключения? Можно ли использовать
    важностную выборку для генерации случайных чисел?
