1.  Сформулируйте задачу генерации дискретной случайной величины с
    заданным распределением вероятностей $p_i = P(X = x_i)$,
    $i = 1, \dots, K$. В чём отличие от генерации непрерывных
    распределений?

2.  Опишите классический способ моделирования дискретной случайной
    величины на основе кумулятивных вероятностей (CDF) с использованием
    бинарного поиска. Запишите алгоритм, оцените временную сложность
    предобработки и генерации.

3.  Сравните классический метод (кумулятивные суммы + бинарный поиск) с
    квантильным методом (линейный поиск) по времени генерации. В каких
    случаях бинарный поиск даёт выигрыш, а в каких --- нет?

4.  Опишите квантильный метод (метод обратной функции распределения) для
    дискретных случайных величин. Запишите алгоритм и оцените его
    временную сложность.

5.  Для каких распределений квантильный метод эффективен, а для каких
    --- нет? Приведите примеры (биномиальное, пуассоновское,
    категориальное с большим $K$).

6.  Можно ли применить квантильный метод для распределений с бесконечным
    носителем (например, распределение Пуассона)? Предложите модификацию
    алгоритма.

7.  В чём заключается метод Уолкера (alias method)? Опишите идею
    алгоритма и его основные этапы (построение таблиц и генерация).

8.  Сравните квантильный метод и метод Уолкера по следующим критериям:
    время предобработки, время генерации, объём памяти, применимость к
    динамически меняющимся распределениям.

9.  В каких случаях метод Уолкера даёт наибольший выигрыш по сравнению с
    квантильным методом? Приведите примеры из машинного обучения
    (например, семплирование из категориального распределения в обучении
    с подкреплением).

10. Опишите, как модифицировать метод Уолкера для распределений с
    бесконечным числом исходов (например, усечённое распределение
    Пуассона). Предложите подход с использованием «хвостовой»
    компоненты.
