## ВХ-3. Тематическое содержание проектной работы

### Тип задач
- Генерация псевдослучайных чисел (LCG, метод вычетов, тестирование).
- Моделирование случайных величин (обратное преобразование, метод Неймана, суперпозиция, Уолкер).
- Оценка интегралов методом Монте-Карло и методы уменьшения дисперсии (выборка по важности, стратификация, выделение главной части).
- Цепи Маркова и MCMC (Метрополис-Гастингс, Гиббс, диагностика сходимости).
- Решение интегральных уравнений Фредгольма II рода (ряд Неймана, оценка по столкновениям).
- Квази-Монте-Карло (последовательности Соболя, Холтона, Ван дер Корпута).
- Моделирование случайных процессов (броуновское движение, пуассоновские процессы, гауссовские процессы).

### Класс ИИС
- Система поддержки принятия решений на основе байесовского вывода.


### Технологии
- Python 3.8+, NumPy, SciPy, Pandas, Matplotlib.
- PyMC, Pyro, TensorFlow Probability (для байесовского вывода).
- PyTorch / TensorFlow (для VAE, MC Dropout).
- Git, Jupyter Notebook.
